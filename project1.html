Executive Summary

Medicare Fee for Service improper payments totaled $31.70 billion in FY 2024, representing a 7.66% improper payment rate. This system introduces the Medicare Fraud Risk Index (MFRI), a framework that predicts which Medicare Part D prescribers are at risk for compliance violations before improper payments occur. The system analyzes 1.38 million Medicare Part D prescribers across five federal databases, generating risk scores that enable investigators to focus resources on the highest risk providers rather than conducting random audits.

Key Achievement: The system processes 26 gigabytes of federal Medicare data covering 1.38 million healthcare providers, achieving 72.46% predictive accuracy in temporal validation. Among the top 1,000 highest risk scores, the system identified 26 providers who were actually excluded by OIG, representing a 33 fold improvement over random selection.

Methodology

Data Integration Framework

This research integrates five authoritative federal datasets spanning 2022 to 2023:

  • Medicare Part D Prescribers by Provider (CMS): 1.38 million providers with 84 variables including claims, costs, and beneficiary demographics
  • Open Payments Database (CMS): Industry payment relationships, amounts, frequency
  • LEIE Exclusions Database (HHS OIG): Exclusion type, date, reinstatement status
  • Medicare Part D by Provider and Drug (CMS): Drug specific prescribing patterns
  • Medicare Part D by Geography and Drug (CMS): Regional benchmarks, peer comparisons

Total data volume: 26 GB

Machine Learning Approach

XGBoost Random Forest SHAP Explainability Temporal Validation Ensemble Methods

The analytical engine combines three complementary machine learning algorithms: Logistic Regression for baseline establishment (AUC: 0.613), Random Forest for non linear pattern detection (AUC: 0.705), and XGBoost for gradient boosting optimization (AUC: 0.724). The final ensemble model combines predictions using weighted voting with a 2:1 XGBoost to Random Forest ratio (66.7% XGBoost, 33.3% Random Forest).

Research Contribution

This project represents a novel application of temporal machine learning to Medicare Part D prescriber risk prediction. The key innovation is the temporal validation methodology: the system trains on 2022 prescribing data and tests predictions against actual 2023 outcomes, demonstrating genuine predictive capability rather than retrospective pattern matching.

The framework advances healthcare program integrity analytics by:

  • Demonstrating that temporal validation can achieve 72.46% predictive accuracy
  • Integrating five federal databases into unified risk assessments
  • Achieving 33 fold improvement over random audit selection
  • Processing 1.38 million providers in 3.7 hours using standard infrastructure

The methodology is documented for reproducibility and the complete source code is publicly available under MIT open source license.

Technical Implementation

Core Algorithm Parameters

  • XGBoost: n_estimators=200, max_depth=6, learning_rate=0.05, scale_pos_weight=50
  • Random Forest: n_estimators=300, max_depth=10, min_samples_split=5
  • Ensemble Weighting: 66.7% XGBoost, 33.3% Random Forest
  • Temporal Validation: Train on 2022 data, test on 2023 data

Key Risk Indicators (SHAP Importance)

  • Total Claims Volume: 9.8% importance
  • Opioid Prescription Share: 9.6% importance
  • Beneficiary Risk Score: 8.6% importance
  • Cost per Claim Ratio: 8.0% importance
  • Brand Drug Preference: 7.8% importance

Validation Procedures

  • Bootstrap Analysis: 1,000 iterations, Mean AUC: 0.725, 95% CI: [0.711, 0.738]
  • Cross Validation: 5 fold, Mean AUC: 0.719, Standard Deviation: 0.021
  • Calibration Testing: Hosmer Lemeshow statistic: 8.32, p value: 0.40
