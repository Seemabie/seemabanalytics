Publications

Methodology preprints

Eight working papers documenting the methods behind the Medicare program integrity portfolio. Each is posted to SSRN and paired with a public code repository that reproduces every numerical claim from public CMS data. Each paper presents anomaly detection for investigative prioritization, not fraud determination.
